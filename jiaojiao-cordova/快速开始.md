# 娇娇 Android应用 - 快速开始指南

## 方法一：使用在线构建服务（最简单）

### 使用Monaca或PhoneGap Build

1. **PhoneGap Build** (Adobe的免费服务)
   - 访问 https://build.phonegap.com/
   - 注册Adobe账号
   - 上传 `娇娇-Android项目.zip`
   - 等待构建完成，下载APK

2. **Monaca** (日本公司的免费服务)
   - 访问 https://monaca.io/
   - 注册账号
   - 创建新项目，上传代码
   - 云端构建APK

## 方法二：本地构建（需要配置环境）

### 步骤1：安装必要软件

1. **Node.js** (必须)
   - 访问 https://nodejs.org/
   - 下载 LTS 版本 (推荐 v18或v20)
   - 安装时勾选"Add to PATH"

2. **Java JDK 17** (必须)
   - 访问 https://adoptium.net/
   - 下载 OpenJDK 17 (LTS)
   - 安装并设置环境变量 `JAVA_HOME`

3. **Android Studio** (必须)
   - 访问 https://developer.android.com/studio
   - 下载并安装
   - 打开SDK Manager，安装：
     - Android SDK Platform 34
     - Android SDK Build-Tools 34
     - Android SDK Command-line Tools
   - 设置环境变量 `ANDROID_HOME`
     - 默认路径: `C:\Users\你的用户名\AppData\Local\Android\Sdk`

### 步骤2：运行构建脚本

1. 解压 `娇娇-Android项目.zip`
2. 双击运行 `build-simple.bat`
3. 等待构建完成（首次构建需要下载依赖，可能需要10-30分钟）
4. APK将生成在 `output/娇娇-v1.0.0-debug.apk`

### 步骤3：安装到手机

**方法A - 直接安装：**
1. 将APK传输到手机
2. 在文件管理器中点击APK
3. 允许"未知来源"安装

**方法B - 使用ADB：**
```bash
adb install output/娇娇-v1.0.0-debug.apk
```

## 方法三：使用Android Studio直接导入

如果你已经安装了Android Studio，可以直接导入项目：

1. 打开Android Studio
2. 选择 "Open an existing project"
3. 选择 `jiaojiao-cordova/platforms/android` 文件夹
4. 等待Gradle同步完成
5. 点击 "Build" -> "Build Bundle(s) / APK(s)" -> "Build APK(s)"
6. APK将生成在 `app/build/outputs/apk/debug/`

## 常见问题

### Q: 双击bat文件后窗口一闪而过？
A: 这通常是因为缺少Node.js或Android SDK。请按照"步骤1"安装必要软件。

### Q: 构建时提示"Gradle sync failed"？
A: 
1. 检查网络连接（需要访问Google服务器）
2. 在Android Studio中点击 "File" -> "Sync Project with Gradle Files"
3. 或者删除 `platforms` 文件夹后重新运行构建脚本

### Q: 如何生成Release版本（用于发布）？
A:
1. 生成签名密钥：
   ```bash
   keytool -genkey -v -keystore jiaojiao.keystore -alias jiaojiao -keyalg RSA -keysize 2048 -validity 10000
   ```
2. 构建Release版本：
   ```bash
   cordova build android --release
   ```
3. 签名APK：
   ```bash
   jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore jiaojiao.keystore platforms/android/app/build/outputs/apk/release/app-release-unsigned.apk jiaojiao
   ```

### Q: 应用图标可以修改吗？
A: 可以。替换 `res/icon/android/` 目录下的图标文件，然后重新构建。

## 技术支持

如果遇到问题，请检查：
1. Node.js是否正确安装: `node -v`
2. Java是否正确安装: `java -version`
3. Android SDK是否正确设置: `echo %ANDROID_HOME%`
4. Cordova是否正确安装: `cordova -v`
